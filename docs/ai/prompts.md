# Registro de Prompts Utilizados

### Técnica: Few-Shot Prompting
*   **Propósito:** Mejorar la generación de código dando contexto previo.
*   **Prompt utilizado:** "Actúa como un desarrollador Frontend. Genera una función pura en JavaScript para filtrar las tareas completadas del LocalStorage en la app TaskFlow."
*   **Resultado:** La IA generó la lógica de filtrado correctamente utilizando el método `.filter()` de los arrays de JavaScript.
