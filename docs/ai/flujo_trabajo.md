# Flujo de Trabajo con IA (Cursor)

## 1. Interfaz y Chat Contextual
He utilizado la generación de código integrada (atajo Ctrl+I). La gran ventaja es que la IA lee el contexto del archivo abierto directamente sin tener que copiar y pegar en el navegador.

## 2. Edición Inline
*   **Uso:** Modificación rápida directamente en el editor.
*   **Ejemplo:** Autocompletado de clases de Tailwind CSS en el HTML y refactorización de funciones en el archivo `app.js`.

## 3. Atajos de Productividad
*   `Ctrl + I`: Generar código inline.
*   `Tab`: Aceptar sugerencias (Ghost text) mientras escribo.
